# Number Guessing Game

A Python command-line game where users guess a randomly generated number.

## Features
- Difficulty levels (Easy, Medium, Hard)
- Limited attempts
- Hint system
- High score tracking
- Play again option

## Technologies Used
- Python
- Random module
- File handling

## Run Locally

```bash
python main.py
```
